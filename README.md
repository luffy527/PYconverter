# File Converter (Python CLI)

Python CLI tool to convert image, document, and media files.

## Features (example)

- Image conversion: PNG, JPG, WEBP, GIF, etc.
- Document conversion: PDF, DOCX, TXT, MD, etc.
- Audio/Video conversion: MP3, WAV, OGG, MP4, etc.
- PDF conversion via external API
- Terminal interface using Rich

## Requirements

- Python 3.10 or higher
- FFmpeg installed
- Pandoc installed
- ChargedAPI account + API key for PDF conversion: https://www.chargedapi.com/  
  → Create an account and generate an API key from the dashboard  
  → Configure the API key in the project
